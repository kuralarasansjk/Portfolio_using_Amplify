Cloud Portfolio Website on AWS Amplify

This project is a personal portfolio website hosted on AWS Amplify. It features a fully automated CI/CD pipeline that integrates with GitHub, so any code changes pushed to the GitHub repository trigger an automatic build and deployment on AWS Amplify. The site is hosted on Amazon S3 with global fast delivery using CloudFront CDN, secured with HTTPS via AWS Certificate Manager, and protected by AWS WAF. Route 53 manages the custom domain, and CloudWatch monitors performance and uptime. The result is a reliable, secure, and up-to-date portfolio that updates instantly whenever you push new code.
